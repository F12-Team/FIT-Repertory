# Проект: «Информационная система хранения и демонстрации результатов проектной деятельности факультета информационных технологий»

В настоящее время есть большое количество задач в рамках проектной деятельности, но нет возможности визуализировать информацию о них. 

Нашей целью является cоздание ресурса для предоставления общих сведений о проектной деятельности на факультете информационных технологий индустриальным партнерам, студентам,  преподавателям и абитуриентам.

Задачи по проекту:
- изучение предметной области;
- проработка основной логики системы;
- проектирование и наполнение базы данных существующими проектами;
- проектирование нового дизайна сайта;
- разработка клиентской части;
- разработка серверной части;
- написание отчетной документации.

С более подробной информацией можно ознакомится в [отчетном документе](https://github.com/F12-Team/FIT-Repertory/blob/master/docs/%D0%94%D0%BE%D0%BA%D1%83%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D1%86%D0%B8%D1%8F.pdf).
